# Low-Cost-EEG-Data-Acquisition-ADS1299IPAGR-
An affordable, highly flexible EEG Data Acquisition Device based on ADS1299IPAGR as ADC, interfaced with ESP32-WROM. 
This project describes the structure, functioning, features and theories of the device. Refer PPT for briefing.
